# Russia Trip Page

This is a website dedicated to documenting and sharing my amazing trip to Russia. It showcases the highlights, experiences, and memories I collected during my journey.

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
## Features
- [x] Interactive map displaying visited locations
- [x] Photo gallery with captions
- [x] Blog-style entries for each day of the trip- [x] Contact form for questions and feedback
- [x] Search functionality to quickly find specific content
## Setup
1. Clone this repository to your local machine.
2. Ensure you have a PHP-enabled server set up (e.g., XAMPP, MAMP, WAMP).
3. Place the cloned files in the appropriate directory (e.g., htdocs for XAMPP).
4. Start your server.
## Usage
1. Open your preferred web browser.
2. Navigate to http://localhost/russia-trip-page.3. Explore the various sections of the website:
   - Browse the interactive map to see visited locations.   - Click on photos in the gallery to view them in full size with captions.
   - Read the blog entries to relive the journey day by day.   - Use the search bar to quickly find specific content.
   - Contact form is available for questions and feedback.
